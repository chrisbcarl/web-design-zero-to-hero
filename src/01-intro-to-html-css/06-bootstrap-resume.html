About Bootstrap

Everything is about the CSS classes.
Start by looking at a template like this one, and see how you can adapt it to fit your needs.
These classes use JavaScript built by the community to quickly and easily add functionality to your website like drop down menu behavior, animations, and even some things like parallax.
Classes can be grouped into the following categories:

Containers

  • container - class for a div to hold other divs
  • row - class for a div to contain a the contents of a row on a web page.
    All of the "sections" you see on this page are contained within rows.
  • column - class for a div to contain the contents of a column.
    if you want text to be spread out into a newspaper style, you would use three divs and give them the class of column.

Sizes or Positions of Containers

  • sm, md, lg, xl- class that you can append to a column so that it appears 'better' on small devices
  • fluid - class that you can append to images, containers, and other elements to make them fill the full width of the page.
  • margin, padding - classes like 'mr-auto' or 'py-auto' allow you to space your elements neatly and give your rows and columns some breathing room.
  • justify-content, align-items - classes allow you to take the contents of a div or take the div itself and center or project them in the way you want!

Nav, Card Elements

  • nav - some classes apply only to navs. You can see our navbar at the top which apppears at all of our webpages, but the nav bar that came with this theme is styled using 'navbar', 'nav-item', and other classes
  • card - a concept of a card came into bootstrap recently thanks to the advent of android. These can be used to hold stylized information in a pleasing way.

Buttons, Font-Awesome Icons, and so much more

      btn - a stylized button that's leaps and bounds prettier than anything HTML5 can offer you.
      primary, secondary, etc. - classes that specify colors based on a palette. If you're a graphic designer, you'll appreciate the ability to customize everything to the color speficiations and relationshjips of your choice.
      fa fa-facebook, etc. - classes that give you the ability to include icons like an Instagram-like camera icon, a YouTube 'like' button, and so many others.
